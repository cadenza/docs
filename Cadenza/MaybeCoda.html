⊟ Remarks

This method is provided to support C# query comprehension expressions on Cadenza.Maybe<T> types.

If self contains no value (Maybe<TSource>.HasValue is false), then Maybe<TResult>.Nothing is returned.

Otherwise, selector(self.Value).ToMaybe() is returned.

	1.Just ().Select (x => x + 1));
Assert.AreEqual (2.Just (),
	from x in 1.Just ()
	select x + 1);

⊟ Returns

A Cadenza.Maybe<TResult> containing Maybe<TResult>.Nothing if either self's or selector(self)'s Maybe<TCollection>.HasValue property is false; otherwise, returns resultSelector(self.Value, selector(self).Value).ToMaybe().

Type Reason
ArgumentNullException

selector is null.

-or-

⊟ Remarks

This method is provided to support C# query comprehension expressions on Cadenza.Maybe<T> types. SelectMany acts as a "null propagation of nullables" for any type, as opposed to just Nullable<T> types.

If self contains no value (Maybe<TSource>.HasValue is false), then Maybe<TResult>.Nothing is returned.

If selector(self) contains no value (Maybe<TCollection>.HasValue is false), then Maybe<TResult>.Nothing is returned.

Otherwise, resultSelector(self.Value, selector(self).Value).ToMaybe() is returned. (This means that if resultSelector returns null, then Maybe<TResult>.Nothing is returned.)

C# Example 
Assert.AreEqual (Maybe<int>.Nothing, 
		5.Just().SelectMany(
			x => Maybe<int>.Nothing,
			(x, y) => x + y));
Assert.AreEqual (Maybe<int>.Nothing,
		from x in 5.Just ()
		from y in Maybe<int>.Nothing
		select x + y);
Assert.AreEqual (9.Just (),
		5.Just().SelectMany(
			x => 4.Just (),
			(x, y) => x + y));
Assert.AreEqual (9.Just (),
		from x in 5.Just ()
		from y in 4.Just ()
		select x + y);
